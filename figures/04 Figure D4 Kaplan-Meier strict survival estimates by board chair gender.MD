Kaplan-Meier survival curves by board chair gender over the first five years after registration, using the strict failure definition (bankruptcy and forced dissolution only; voluntary discontinuations censored). Firms with uncertain gender classification are excluded from this figure only (they are included in all regression models). Log-rank test: χ²(1) = 0.10, p = .748. n = 54,391; strict failures = 4,269.
