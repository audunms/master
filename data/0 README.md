This folder contains a censored version of the source data that can be used as input when running the complete Stata commands. Organisation number is censored, as well as and objective and activity (these are not openly availble for deregistered firms). Due to size limitations in Github, approx. half of the rows in the Excel sheet were removed before uploading. The full and uncensored data is available upon request to audunms@gmail.com.

Furthermore, a detailed description of the data pre-processing and Excel commands for keyword search os also found in this folder.
