# master
Code and Stata output related to master thesis by Audun Minnesjord Solheim

This repository has several supplementary files for the master thesis. Files are organized in different folders:

1. **Stata** Here is the complete Stata code used in the thesis, as well as the complete Stata output from running the Stata code. The code can be run in one go (Appendix E). 
2. **Tables**. Additional tables not included in the main thesis document (Appendix B, D and I).
3. **Figures**. Additional figures not included in the main thesis document (Appendix D).
4. **Data**. The source data, from Proff and The Brønnøysund Register Centre, used in the thesis. Also detailed descriptions of the data preprocessing steps and Excel search formulas are found here (Appendix G).

All tables, figures and Stata output are based on own calculations and data from The Brønnøysund Register Centre and Proff AS.
